# Notebook Status

These notebooks are lightweight analysis layers on top of JSON files written by the scripts. They are optional helpers; the final submitted statistics and figures are generated by `scripts/build_final_assets.py` under `results/final_clean/`.

Current notebooks:

- `01_project_overview.ipynb`: project-level dashboard for goals, implemented pieces, current results, and next steps.
- `02_baseline_comparison.ipynb`: loads a baseline summary and compares the heuristic controllers across regimes.
- `03_dqn_training_analysis.ipynb`: loads a DQN summary, inspects DQN training behavior, and compares DQN against baselines.

Recommended workflow:

```bash
python3 scripts/run_baselines.py --config configs/default.yaml
python3 scripts/train_dqn.py --config configs/default.yaml
python3 scripts/tune_dqn.py --config configs/default.yaml
```

Then open the notebooks for exploratory inspection. For the clean final report and presentation figures, use `results/final_clean/final_statistics.md` and `results/final_clean/figures/`.
